v3.0.0 - 2026-02-04

PySpOS 3.0.0 测试版

• 完善了zzlsb猜数字游戏
• 修复若干bug
• 实现了简易的recovery和日志处理
⚠ 此release为测试版，可能会出现bug，你可以将bug提交到Issues.

兼容性要求

  • Python 3.7 或更高版本
  • Windows 7 及以上 / Linux / macOS
  • 至少 50MB 可用磁盘空间
  • 支持双槽位更新（A/B 槽位）
  • 需要网络连接用于 OTA 更新

回滚指南

如果更新后遇到问题，可以回滚到之前的版本：

  • 在 PySpOS 中输入 recovery 命令进入恢复模式
  • 在恢复模式中输入 rollback 命令
  • 确认回滚操作
  • 重启 PySpOS 以应用更改

注意：回滚操作会切换到另一个槽位，请确保另一个槽位包含有效的系统版本。
