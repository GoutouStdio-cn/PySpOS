兼容性要求

  • Python 3.7 或更高版本
  • Windows 7 及以上 / Linux / macOS
  • 至少 50MB 可用磁盘空间
  • request 库（用于抓取云端 ota 内容）

回滚指南

如果更新后遇到问题，可以回滚到之前的版本：

  • 在 PySpOS 中输入 recovery 命令进入恢复模式
  • 在恢复模式中输入 ota_rollback 命令
  • 确认回滚操作
  • 重启 PySpOS 以应用更改

注意：回滚操作会切换到另一个槽位，请确保另一个槽位包含有效的系统版本。（如没有文件，系统可能会变砖。）
