发展历程

2022
"MP4"操作系统：狗头OS

（以下我指JiTianYu）GoutouStdio 工作室正式创立，我看到网上的狗头os项目，正式开始了"研发"，最后得到5000粉丝

2023
抄袭与合作

开始抄袭别人的代码，不过我抄的代码的那个创作者不知道咋的就认识了，然后我俩就开始互相学习（

2024
VimdropOS诞生，成功被激励

VimdropOS（由 ViudiraTech 打造）正式发布，我们基于它制作了第一版 SpaceOS，开启了新的篇章。

2025
UEFI 研究与 PySpOS

开始深入研究 UEFI，同时利用 ViudiraTech 的 Uinxed-Kernel 项目，正式开始研发 PySpOS。

2026
自研操作系统

UEFI 终于完全理解，开始完全自研操作系统。目前项目已完成约二分之一，正在稳步推进中。（用户态没做好）

技术优势

高性能内核

从 PySpOS 的 Python 模拟到 SpaceOS 系列的混合内核架构，我们始终坚持高性能设计原则，确保系统流畅运行。

高安全性

Ring 0-3 分层权限管理，结合自研文件系统与安全引导机制，全方位保障系统安全。

模块化设计

清晰的模块划分与接口定义，支持 OTA 在线更新，方便功能扩展与维护。

开源精神

我们致力于开源社区建设，所有核心代码均在 GitHub 开源，欢迎开发者共同参与。

持续创新

从 SPF 文件格式到 SPFS 文件系统，我们不断探索操作系统技术前沿。

社区驱动

由 GoutouStdio 工作室主导，社区开发者共同维护，打造真正属于开发者的操作系统。
