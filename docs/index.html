欢迎使用 PySpOS

一个使用 Python 制作的模拟操作系统，方便让新手理解操作系统基本操作

下载更新 下载源代码

系统特性

跨平台支持

支持 Windows 7 及以上、Linux 和 MacOS 系统

OTA 更新

支持从云端下载和安装系统更新

Bootloader 管理

支持 Bootloader 锁定/解锁和 ROOT 权限管理

双槽位系统

A/B 槽位设计，支持安全更新和回滚

应用系统

支持运行自定义 Python 应用和 SPF 可执行文件

恢复模式

内置恢复模式，可修复系统问题

支持平台

Windows 7 及以上 推荐
Linux
MacOS

快速开始

1

克隆仓库

git clone https://github.com/GoutouStdio-cn/PySpOS.git
2

进入目录

cd PySpOS
3

运行系统

python main.py
