产品介绍

什么是 SpaceOS 7？

SpaceOS 7 是我们完全自研的混合内核架构操作系统。从 2026 年开始，我们完全理解了 UEFI 技术，开始完全自研操作系统。目前项目已完成约二分之一，正在稳步推进中。

项目起源

SpaceOS 7 的研发始于 2026 年，这是我们在完全理解 UEFI 技术后启动的完全自研项目。与之前的版本不同，SpaceOS 7 从零开始构建，不依赖任何外部代码，真正实现了完全自研。

设计理念

SpaceOS 7 的设计理念是"完全自研、混合内核、极致性能"。我们相信，只有完全自研才能真正掌握核心技术，实现真正的创新。混合内核架构结合了微内核和宏内核的优点，既保证了系统的稳定性，又提供了卓越的性能。

核心特性

混合内核架构

全手搓自研混合内核，保证系统稳定性和性能。SpaceOS 7 的混合内核结合了微内核和宏内核的优点，既保证了系统的稳定性，又提供了卓越的性能。

  • 微内核组件用户空间化
  • 关键服务内核空间运行
  • 高效的 IPC 机制
  • 完善的中断处理

内核重构

中间引导和内核完全重构，全部手搓自研。SpaceOS 7 的内核从零开始构建，不依赖任何外部代码，真正实现了完全自研。

  • 完全自研的引导程序，并没有使用Limine和grub这样的开源引导程序
  • 自研的内核架构，抛弃Uinxed-Kernel内核转为纯自研
  • 优化的内存管理，支持伙伴分配器
  • 高效的进程调度，支持进程间IPC，支持用户态

Ring 3 桌面环境

桌面环境运行在 Ring 3，提升系统安全性。通过将桌面环境运行在用户空间，即使桌面环境崩溃也不会影响内核。

  • 桌面环境用户空间运行，稳定性更高
  • 内核空间与用户空间隔离
  • 更高的系统安全性（因为有ring3）
  • 更好的稳定性（因为桌面环境一崩内核不会炸）

SMP 多核支持

实现 SMP (Symmetric Multi-Processing) 多核支持。SpaceOS 7 完美支持多核处理器，能够充分利用多核性能。

  • 多核调度支持
  • 多核同步机制
  • 多核内存管理
  • 多核中断处理

xHCI 控制器

完整支持 xHCI (eXtensible Host Controller Interface)。SpaceOS 7 提供了完整的 xHCI 驱动，支持 USB 3.0 及以上版本。

  • USB 3.0 支持
  • USB 2.0 向后兼容
  • 多设备支持
  • 高带宽传输

空间 API

支持空间 API，提供更丰富的系统调用接口。空间 API 是我们自研的系统调用接口，提供了更强大的功能。

  • 丰富的系统调用
  • 高性能 IPC
  • 灵活的通信机制
  • 安全的权限控制

性能提升

相比 SpaceOS 6，性能大幅提升，系统更加流畅。通过优化内核和各种组件，SpaceOS 7 在各个方面都有显著的性能提升。

  • 更快的系统启动
  • 更快的应用启动
  • 更快的文件操作
  • 更流畅的用户体验

技术细节

混合内核架构

SpaceOS 7 采用创新的混合内核架构：

  • 微内核组件：核心服务运行在用户空间，包括文件系统、网络协议栈等
  • 宏内核组件：关键驱动和子系统运行在内核空间，包括中断处理、进程调度、内存管理等
  • IPC 机制：高效的进程间通信机制，确保用户空间和内核空间的高效协作
  • 权限管理：严格的权限控制，确保系统安全

这种架构既保证了系统的稳定性（用户空间组件崩溃不会影响系统），又提供了卓越的性能（关键组件在内核空间运行）。

引导程序

SpaceOS 7 实现了完全自研的引导程序，从零开始手写 UEFI 库：

  • 纯手写 UEFI 库：完全自研的 UEFI 库，不依赖任何外部代码（除efi.h，那是基本的），深入理解 UEFI 规范
  • 高半地址内核加载：支持高半地址（Higher Half）内核加载，优化内存布局和性能
  • UEFI 引导支持：完整的 UEFI 引导支持，支持现代计算机的启动方式

引导程序经过精心设计，确保在各种硬件上都能稳定启动。

内存管理

SpaceOS 7 实现了先进的内存管理机制：

  • 分页管理：支持 4KB 和 2MB 大页
  • 虚拟内存：完整的虚拟内存支持
  • 内存保护：硬件辅助的内存保护
  • 内存压缩：可选的内存压缩，提高内存利用率
  • NUMA 支持：支持非统一内存访问架构

内存管理经过深度优化，确保在各种场景下都能高效运行。

进程调度

SpaceOS 7 实现了先进的进程调度算法：

  • CFS (Completely Fair Scheduler)：完全公平调度器
  • 实时调度：支持实时进程调度
  • 多核调度：支持多核处理器的负载均衡
  • 节能调度：根据系统负载自动调整 CPU 频率

调度器经过精心设计，确保公平性和效率。

设备驱动

SpaceOS 7 提供了丰富的设备驱动支持：

  • 存储驱动：支持 IDE、AHCI、NVMe、USB、SCSI 等各种存储设备
  • 显示驱动：支持各种显卡，包括集成显卡和独立显卡
  • 网络驱动：支持各种有线和无线网卡
  • 音频驱动：支持各种声卡
  • 输入驱动：支持键盘、鼠标、触摸板、触摸屏等
  • USB 驱动：完整支持 xHCI，支持 USB 3.0 及以上版本

所有驱动都经过严格测试，确保稳定性和兼容性。

空间 API

空间 API 是我们自研的系统调用接口：

  • 丰富的系统调用：提供各种系统调用，满足各种需求
  • 高性能 IPC：高效的进程间通信机制
  • 灵活的通信机制：支持消息队列、共享内存、信号量等
  • 安全的权限控制：严格的权限控制，确保系统安全

空间 API 经过精心设计，既强大又易用。
