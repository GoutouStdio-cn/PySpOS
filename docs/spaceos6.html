项目介绍

什么是 SpaceOS 6？

SpaceOS 6 是我们推出的高性能桌面操作系统，基于 Uinxed-Kernel（由 ViudiraTech 打造）进行深度定制和优化。它集成了我们自研的多项核心技术，为用户提供流畅、稳定的桌面体验。

从 2024 年开始研发基于 Uinxed-Kernel 的 SpaceOS，我们不断迭代优化，最终推出了 SpaceOS 6 这一成熟稳定版本。

Uinxed-Kernel 内核

SpaceOS 6 基于 Uinxed-Kernel 构建，这是由 ViudiraTech 开发的 64 位宏内核。Uinxed-Kernel 运行在 CPU Ring 0 最高特权级，提供完整的系统调用接口、进程调度、内存管理和文件系统支持。作为宏内核架构，所有核心组件（进程管理、内存管理、文件系统、设备驱动等）都运行在内核空间，确保高效的系统性能和低延迟的硬件访问。

SPUI 1.0 界面

SpaceOS User Interface 1.0 是我们自主研发的桌面环境，提供现代化的用户交互体验。界面简洁直观，操作流畅高效，不适合日常办公和娱乐使用。

硬件支持

SpaceOS 6 支持现代硬件技术，包括 AHCI、NVMe 存储控制器，SB16 声卡，以及 JPG 图片硬件加速。分区表支持让多系统部署更加灵活。

核心特性

SPUI 1.0 桌面

自研的 SpaceOS User Interface 1.0，提供现代化的桌面体验。支持窗口管理、任务栏、开始菜单等功能。

SPF Parser 2.0

支持 SPF (SpaceOS Format) 可执行文件的解析和执行。SPF 2.0 版本带来更好的性能和更多系统调用支持。

JPG 图片解析

原生支持 JPG 图片格式的解析和显示，无需第三方库。支持多种尺寸和压缩质量。

SB16 声卡支持

完整支持 SB16 声卡，提供高质量的音频输出。支持 PCM 音频播放和 MIDI 音乐合成。

分区表与存储

支持分区表、AHCI、NVMe 等现代存储技术。支持多分区管理和引导加载器配置。

SPFS 2.0 文件系统

自研的高性能文件系统 SPFS 2.0，提供卓越的文件操作性能。支持文件权限管理和日志功能。

技术规格

Uinxed-Kernel

由 ViudiraTech 开发的 64 位宏内核，运行在 Ring 0 最高特权级。提供完整的系统调用、进程调度、内存管理和文件系统支持。

宏内核架构

所有核心组件（进程管理、内存管理、文件系统、设备驱动）都运行在内核空间，确保高效的系统性能和低延迟的硬件访问。

Ring 0 特权级

内核运行在 CPU 最高特权级 Ring 0，拥有对硬件的完全访问权限，实现高效的系统资源管理和硬件控制。

内存管理

支持分页内存管理，提供内存保护机制。优化内存分配算法，减少碎片化。

存储支持

支持 IDE、AHCI、NVMe 存储接口。支持 MBR 和 GPT 分区表格式。

音频系统

SB16 兼容音频驱动，支持 DMA 传输。提供音频混合和音量控制功能。

系统要求

处理器

x86 架构处理器，支持 Pentium 及以上

内存

最低 64MB，推荐 128MB 及以上

存储

至少 500MB 可用空间

显卡

VESA 兼容显卡，支持 800x600 分辨率
